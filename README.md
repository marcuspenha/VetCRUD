# Sistema CRUD para Clínica Veterinária

Este projeto implementa um sistema de gerenciamento para uma clínica veterinária, permitindo cadastrar, visualizar, atualizar e excluir informações relacionadas a animais, tutores e serviços. O objetivo é aplicar conceitos de CRUD e Programação Orientada a Objetos de forma prática e organizada.

## Funcionalidades

### Animais
- Cadastro de animais
- Listagem de animais
- Atualização de dados
- Exclusão

### Tutores
- Cadastro de tutores
- Associação entre tutor e animais
- Edição de informações
- Remoção

### Serviços
- Registro de consultas e procedimentos
- Atualização de serviços
- Remoção

### Relatórios
- Relatório geral com dados de animais, tutores e serviços

## Tecnologias e Paradigma

- Java
- Programação Orientada a Objetos (POO)
- Menu interativo pelo console

## Como executar

1. Clone o repositório:
   git clone https://github.com/marcuspenha/VetCRUD.git

2. Acesse a pasta:
   cd seu-repositorio

3. Compile os arquivos:
   javac *.java

4. Execute o programa principal:
   java Main

## Objetivo do Projeto

Demonstrar de forma prática:
- Estruturação de classes
- Operações CRUD completas
- Encapsulamento
- Organização e separação de responsabilidades
- Construção de um sistema simples em Java
